# 💬 Real-Time Chat Application (MERN + Socket.IO)

## 🚀 Overview

This is a full-stack real-time chat application built using the MERN Stack and Socket.IO.  
The application enables users to communicate instantly through real-time messaging, media sharing, and live online status updates with a modern and responsive user interface.

---

# ✨ Features

## 🔐 Authentication System
- User Signup & Login
- Secure Authentication & Session Handling
- Protected Routes

## 💬 Real-Time Messaging
- One-to-One Real-Time Chat
- Instant Message Delivery using Socket.IO
- No Page Reload Required

## 📁 Media Sharing
Users can send and receive:
- Text Messages
- Images
- Videos
- Files

## 🟢 Online Status Indicator
- Live Online/Offline User Status
- Real-Time User Activity Tracking

## 👤 User Profiles
- Update Profile Picture
- Edit Name
- Update Bio ("About" Section)

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Socket.IO Client
- CSS / Tailwind CSS

## Backend
- Node.js
- Express.js
- Socket.IO

## Database
- MongoDB

---

# ⚙️ How It Works

1. Users create an account or log in
2. After authentication, users are redirected to the main chat interface
3. The system fetches all registered users
4. Users can:
   - Start Conversations
   - Send Real-Time Messages
   - Share Media & Files
5. Socket.IO ensures:
   - Instant Communication
   - Real-Time Online Status Updates
